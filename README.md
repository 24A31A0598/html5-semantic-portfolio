# HTML5 Semantic Portfolio Website

## Project Overview
This project is a multi-page personal portfolio website developed using modern HTML5 semantic elements and accessibility best practices.

The website follows WCAG accessibility guidelines and includes semantic page structure, ARIA support, SEO-friendly metadata, and an accessible contact form.

## Features

- Semantic HTML5 structure
- Multi-page website
- Accessible navigation
- ARIA labels and roles
- SEO optimized meta tags
- Keyboard accessible contact form
- Responsive layout foundation
- Lighthouse Accessibility and SEO friendly

## Pages

- Home (`index.html`)
- About (`about.html`)
- Projects (`projects.html`)
- Contact (`contact.html`)

## Technologies Used

- HTML5
- CSS3

## Accessibility Features

- Semantic tags
- Skip navigation link
- Keyboard navigation support
- Form labels
- ARIA attributes
- Focus indicators
